# Задачи по теме: Цикл FOREACH в PHP (20 вариантов)

## Вариант 1
Переберите ассоциативный массив пользователя: ['name' => 'Иван', 'email' => 'ivan@mail.ru', 'role' => 'admin']. Выведите пары «ключ: значение» в формате «name: Иван» и т.д.

## Вариант 2
Удвойте каждый элемент числового массива [1, 2, 3, 4, 5], используя foreach по ссылке. После цикла вызовите unset($v). Выведите массив — должно быть [2, 4, 6, 8, 10].

## Вариант 3
Создайте новый массив из ассоциативного ['a' => 1, 'b' => 2]: ключи преобразуйте в верхний регистр (strtoupper). foreach, новый массив $result. Выведите $result.

## Вариант 4
Переберите многомерный массив [['name' => 'A', 'age' => 20], ['name' => 'B', 'age' => 25]]. Вложенный foreach: выведите для каждого «name, age лет».

## Вариант 5
Массив строк ['PHP', 'Python', 'Java']. foreach выведите каждый элемент с номером: «1. PHP», «2. Python», «3. Java» (используйте ключ или счётчик).

## Вариант 6
Объедините все значения массива ['Hello', ' ', 'World'] в одну строку через foreach (конкатенация в переменную $s). Выведите $s.

## Вариант 7
Ассоциативный массив настроек: ['theme' => 'dark', 'lang' => 'ru']. Сформируйте строку вида «theme=dark&lang=ru» с помощью foreach. Не используйте http_build_query.

## Вариант 8
Найдите сумму всех значений массива [10, 20, 30, 40] с помощью foreach. Выведите 100.

## Вариант 9
Переберите массив и выведите только элементы с чётным индексом. Массив [0, 1, 2, 3, 4, 5]. Вывод: 0, 2, 4. В foreach проверяйте $key % 2 === 0.

## Вариант 10
Массив объектов (stdClass) с полем name. Создайте массив имён. foreach по объектам, в новый массив добавляйте $obj->name. Выведите массив имён.

## Вариант 11
Объедините ключи и значения массива ['x' => 1, 'y' => 2, 'z' => 3] в строки «x=1», «y=2», «z=3». Выведите каждую с новой строки.

## Вариант 12
Увеличьте каждый элемент массива [1, 2, 3] на 10 через foreach по ссылке. unset после цикла. Выведите массив [11, 12, 13].

## Вариант 13
Массив слов ['яблоко', 'банан', 'апельсин']. Выведите список <ul><li>...</li></ul> (можно просто строки "• яблоко" и т.д.) с помощью foreach.

## Вариант 14
Найдите максимальное значение в массиве [3, 7, 2, 9, 1] с помощью foreach (без max()). Переменная $max, первое значение — первый элемент.

## Вариант 15
Переберите массив [1, 2, 3, 4, 5] и сформируйте строку «1-2-3-4-5» (через дефис). Используйте foreach и конкатенацию, без implode.

## Вариант 16
Ассоциативный массив: ключи — ID, значения — названия. Выведите таблицу: «ID | Название» и строки для каждой пары. Минимум 3 элемента.

## Вариант 17
Массив с дубликатами [1, 2, 2, 3, 3, 3]. Подсчитайте количество вхождений каждого значения (результат: массив [1=>1, 2=>2, 3=>3]). Используйте foreach и новый массив счётчиков.

## Вариант 18
Переберите массив строк и замените каждую на её длину (результат — массив длин). Используйте foreach по ссылке. Исходный ['ab', 'test', 'x']. Результат [2, 4, 1].

## Вариант 19
Многомерный массив: ключ — категория, значение — массив товаров. Вложенный foreach: выведите «Категория: X», затем список товаров под ней.

## Вариант 20
Массив [5, 10, 15, 20]. foreach: выведите только те элементы, которые больше 10. Результат: 15, 20. В теле if ($value > 10) echo ...
