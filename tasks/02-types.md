# Задачи по теме: Типы данных PHP (20 вариантов)

## Вариант 1
Создайте переменные всех скалярных типов (int, float, string, bool). Выведите их через `var_dump()` в одну строку с комментариями.

## Вариант 2
Напишите функцию `multiply(float $a, float $b): float`, возвращающую произведение. Вызовите с целыми и дробными аргументами, выведите результаты.

## Вариант 3
Реализуйте функцию `processValue(int|string $value): string`, возвращающую строковое представление (число преобразовать в строку, строку вернуть как есть).

## Вариант 4
Создайте переменную `$data`. Поочерёдно присвойте ей int, string, array, null. После каждого присваивания выведите тип через `gettype()`.

## Вариант 5
Напишите функцию `isPositive(int|float $n): bool`, возвращающую true, если число больше 0.

## Вариант 6
Создайте объект `stdClass` с полями `name` и `age`. Присвойте значения. Выведите оба свойства.

## Вариант 7
Реализуйте функцию `describeType(mixed $value): string`, возвращающую строку вида «Передано значение типа: {тип}».

## Вариант 8
Объявите переменную `$items` как массив из трёх строк. Выведите количество элементов и второй элемент. Укажите тип параметра в комментарии.

## Вариант 9
Напишите функцию `concat(string $a, string $b): string`, возвращающую конкатенацию. Вызовите с разными строками.

## Вариант 10
Создайте переменные: целое число, строку с числом "42", булево. Сравните первую и вторую через `==` и `===`. Выведите результаты.

## Вариант 11
Реализуйте функцию `toInt(mixed $value): int`, возвращающую (int) значение. Проверьте на строке "100", float 3.14, bool true.

## Вариант 12
Объявите массив `$config` с ключами 'host', 'port', 'debug' (значения: string, int, bool). Выведите каждый элемент с указанием типа.

## Вариант 13
Напишите функцию `nullableInt(?int $x): ?int`, возвращающую переданное число или null. Вызовите с числом и с null.

## Вариант 14
Создайте переменную `$mixed` и присвойте по очереди: число, строку, массив. После каждого присваивания выведите `var_export($mixed)`.

## Вариант 15
Реализуйте функцию `sumOrConcat(int|string $a, int|string $b): int|string`: если оба int — сумма, иначе конкатенация строк.

## Вариант 16
Объявите переменную `$result` и присвойте результат сравнения (например, 5 > 3). Выведите значение и тип. Повторите для выражения с строками.

## Вариант 17
Напишите функцию `isEmpty(mixed $value): bool`, возвращающую true для null, пустой строки, пустого массива, 0 (на ваш выбор логики).

## Вариант 18
Создайте массив из пяти целых чисел. Выведите тип массива и тип первого элемента через gettype().

## Вариант 19
Реализуйте функцию `ensureString(mixed $value): string`, приводящую значение к строке (число, bool, null — в читаемую строку).

## Вариант 20
Объявите переменную `$id` типа int и присвойте значение. Затем присвойте строку "123". Выведите тип до и после. Продемонстрируйте динамическую типизацию.
