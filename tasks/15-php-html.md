# Задачи по теме: Использование PHP в связке с HTML (20 вариантов)

## Вариант 1
Создайте HTML-страницу с таблицей <table>. Строки генерируйте из массива данных (например, массив из 3 элементов с полями name, age). Выведите строки через foreach. Используйте htmlspecialchars для вывода.

## Вариант 2
Используйте альтернативный синтаксис if/endif: если переменная $hasError истинна, выведите блок <div class="error">Ошибка</div>, иначе — блок с сообщением «Всё хорошо». Выведите оба варианта (переключите $hasError).

## Вариант 3
Форма с предзаполнением: переменные $username и $email (симулируйте из $_POST или задайте). В <input> выведите value="<?= htmlspecialchars($username ?? '') ?>". То же для email. Кнопка отправки.

## Вариант 4
Шаблон карточки товара: переменные $name, $price, $discount (может быть 0). Выведите название и цену. Условие: если $discount > 0, выведите «Скидка: {discount}%» и цену со скидкой. Используйте альтернативный синтаксис.

## Вариант 5
Список <ul>: массив $items из 5 строк. Выведите <li> для каждого элемента с экранированием (htmlspecialchars). Без нумерации, просто список.

## Вариант 6
Условный вывод: переменная $isLoggedIn. Если true — выведите «Привет, {username}!» и ссылку «Выйти». Если false — ссылку «Войти». Используйте if/else с альтернативным синтаксисом.

## Вариант 7
Таблица с заголовками: первый ряд <th> — Name, Age, City. Данные из массива пользователей (2–3 элемента). Второй и последующие ряды — foreach по массиву. Все значения через htmlspecialchars.

## Вариант 8
Форма выбора: переменная $selectedId (например, 2). Массив опций [1 => 'Опция 1', 2 => 'Опция 2', 3 => 'Опция 3']. В <select> выведите <option> для каждого, атрибут selected для элемента с ключом $selectedId.

## Вариант 9
Блок с циклом: массив $tags = ['PHP', 'HTML', 'CSS']. Выведите каждый тег как <span class="tag">{tag}</span> через foreach. Между элементами можно запятую или пробел.

## Вариант 10
Условные классы: переменная $status ('success', 'error', 'warning'). Выведите <div class="alert alert-{status}">Сообщение</div>. Используйте экранирование для $status.

## Вариант 11
Пустой массив: $items = []. Выведите «Список пуст» через if (empty($items)), иначе — список <ul><li>...</li></ul>. Продемонстрируйте оба варианта (с пустым и непустым массивом).

## Вариант 12
Форма с методом POST. Поля: name, email. После «отправки» (симуляция: проверка REQUEST_METHOD === 'POST') выведите «Получено: имя, email» выше формы. Значения полей предзаполните из $_POST с htmlspecialchars.

## Вариант 13
Вывод числа с разделителем: переменная $count = 12345. Выведите «Найдено записей: 12 345» (number_format). В HTML внутри абзаца.

## Вариант 14
Чекбоксы: массив $selected = [1, 3]. Массив опций [1 => 'A', 2 => 'B', 3 => 'C']. Для каждой опции выведите <input type="checkbox"> с name="ids[]", value и checked, если id в $selected. Экранируйте вывод.

## Вариант 15
Три состояния: переменная $state (0 — нет данных, 1 — загрузка, 2 — готово). Выведите разный контент: «Нет данных», «Загрузка...», «Данные загружены» через if/elseif/else с альтернативным синтаксисом.

## Вариант 16
Ссылки из массива: $links = [['url' => 'https://php.net', 'text' => 'PHP'], ...]. Выведите <a href="{url}">{text}</a> для каждого. url экранируйте для атрибута (htmlspecialchars), text — для содержимого.

## Вариант 17
Изображение: переменные $src, $alt. Выведите <img src="<?= htmlspecialchars($src) ?>" alt="<?= htmlspecialchars($alt) ?>">. Если $src пустой — выведите placeholder или текст «Нет изображения».

## Вариант 18
Вложенный цикл: массив категорий, каждая категория имеет массив товаров. Выведите <h3>{категория}</h3> и <ul> с товарами внутри. Минимум 2 категории по 2 товара.

## Вариант 19
Скрытое поле: переменная $token = 'abc123'. В форме выведите <input type="hidden" name="csrf_token" value="<?= htmlspecialchars($token) ?>">. Кнопка отправки.

## Вариант 20
Пагинация (только разметка): переменные $currentPage = 2, $totalPages = 5. Выведите «Назад» (ссылка или текст), номера страниц 1 2 [3] 4 5 (текущая без ссылки или выделена), «Вперёд». Текущую страницу выводите без ссылки, остальные — как ссылки с параметром ?page=N.
