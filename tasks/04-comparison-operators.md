# Задачи по теме: Операторы сравнения PHP (20 вариантов)

## Вариант 1
Сравните числа 5 и "5" с помощью == и ===. Выведите оба результата и объясните разницу в одном комментарии.

## Вариант 2
Используйте оператор <=> (spaceship) для сравнения пар (3, 7), (7, 3), (5, 5). Выведите результаты и поясните: -1, 0, 1.

## Вариант 3
Напишите функцию `compare(int|float $a, int|float $b): string`, возвращающую "больше", "меньше" или "равно" на основе оператора <=>.

## Вариант 4
Проверьте: что выведет `var_dump(0 == "abc")` и `var_dump(null == false)`? Напишите код и закомментируйте, почему для форм лучше использовать ===.

## Вариант 5
Даны переменные `$age` и константа `MIN_AGE = 18`. Выведите true/false для условий: age >= MIN_AGE, age < MIN_AGE, age === 18.

## Вариант 6
Сравните две строки `$a` и `$b` с помощью == и ===. Присвойте им значения "hello" и "HELLO". Выведите результаты. Добавьте сравнение через strcasecmp (результат вывести).

## Вариант 7
Используйте операторы != и !== для пар (10, "10") и (null, false). Выведите все четыре результата.

## Вариант 8
Напишите функцию `inRange(int $value, int $min, int $max): bool`, возвращающую true, если value между min и max включительно. Используйте только операторы сравнения.

## Вариант 9
Для чисел с плавающей точкой 0.1 + 0.2 и 0.3 сравните через ==. Выведите результат. Реализуйте «безопасное» сравнение: abs((0.1+0.2) - 0.3) < 0.00001.

## Вариант 10
Отсортируйте массив [3, 1, 4, 1, 5] по убыванию, используя usort и оператор <=> в функции сравнения. Выведите результат.

## Вариант 11
Сравните пустую строку "" с false, с 0, с null через ==. Выведите три результата. Повторите с ===.

## Вариант 12
Напишите выражение: проверка, что переменная `$status` равна строке "active" (используйте строгое сравнение). Выведите результат проверки.

## Вариант 13
Для переменных `$a` и `$b` (числа) выведите результат всех шести операторов: <, >, <=, >=, ==, ===. Инициализируйте a=5, b=5.

## Вариант 14
Используйте <=> для сортировки массива строк по длине (usort с функцией, сравнивающей strlen($a) <=> strlen($b)).

## Вариант 15
Проверьте: (1 == true), (0 == false), ("1" == true). Выведите результаты. Объясните в комментарии риски при сравнении типов.

## Вариант 16
Напишите функцию `maxOfThree($a, $b, $c)`, возвращающую максимальное из трёх чисел, используя только операторы сравнения и условные выражения.

## Вариант 17
Сравните два массива [1,2,3] и [1,2,3] через == и ===. Выведите результаты. Затем измените порядок во втором — [3,2,1] — и сравните снова с первым.

## Вариант 18
Для переменной `$score` (число от 0 до 100) выведите результат сравнений: score >= 90, score >= 70, score < 70. Используйте одну переменную, три echo.

## Вариант 19
Реализуйте «безопасное сравнение» двух float: функция `floatEquals(float $a, float $b, float $epsilon = 1e-9): bool`.

## Вариант 20
Используйте операторы <, <=, >, >= для сравнения строк "apple" и "banana". Выведите результаты (лексикографическое сравнение).
