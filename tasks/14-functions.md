# Задачи по теме: Функции в PHP (20 вариантов)

## Вариант 1
Напишите функцию `calculateCircle(float $radius): array`, возвращающую массив с ключами 'area' и 'circumference' (площадь и длина окружности). Формулы: πr² и 2πr. Вызовите для радиуса 5 и выведите результат.

## Вариант 2
Создайте функцию с параметром по умолчанию: `greet(string $name, string $greeting = 'Привет')`. Верните строку «$greeting, $name!». Вызовите с одним аргументом и с двумя. Выведите оба результата.

## Вариант 3
Используйте arrow function в array_map: удвойте каждый элемент массива [1, 2, 3, 4, 5]. Результат [2, 4, 6, 8, 10]. Выведите.

## Вариант 4
Реализуйте функцию `mergeStrings(string ...$parts): string`, объединяющую все переданные аргументы через пробел. Вызовите mergeStrings('Один', 'Два', 'Три') и выведите «Один Два Три».

## Вариант 5
Функция `isEven(int $n): bool` — возвращает true, если число чётное. Вызовите для 4 и 5, выведите результаты.

## Вариант 6
Напишите анонимную функцию, сохраните в переменную $square. Вызов $square(5) должен вернуть 25. Выведите $square(5) и $square(10).

## Вариант 7
Функция с опциональным параметром: `formatPrice(float $price, string $currency = '₽'): string`. Верните строку «{price} {currency}». Вызовите с одним и с двумя аргументами.

## Вариант 8
Переменное количество аргументов: функция `product(...$numbers)` возвращает произведение всех переданных чисел. product(2, 3, 4) → 24. Выведите.

## Вариант 9
Arrow function: переменная $isAdult = fn($age) => $age >= 18. Вызовите для 17 и 20. Выведите true/false.

## Вариант 10
Функция `getMax(int $a, int $b): int` возвращает большее из двух. Реализуйте без встроенного max (через if или тернарный оператор). Выведите getMax(10, 20).

## Вариант 11
Замыкание: функция возвращает анонимную функцию, которая добавляет к аргументу фиксированное число. $addFive = createAdder(5); $addFive(10) → 15. Реализуйте createAdder и выведите результат.

## Вариант 12
Функция с типизацией: `divide(float $a, float $b): float`. Верните $a / $b. Обработайте деление на ноль (верните 0 или выбросьте исключение — на выбор). Вызовите divide(10, 3) и выведите.

## Вариант 13
array_map с обычной функцией: напишите функцию double($x), возвращающую 2*$x. Примените к [1, 2, 3]. Выведите массив [2, 4, 6].

## Вариант 14
Функция `repeat(string $s, int $n): string` возвращает строку $s, повторённую $n раз (без str_repeat — циклом). Выведите repeat('ab', 3) → 'ababab'.

## Вариант 15
Параметры по умолчанию: функция `createUser(string $name, string $role = 'user', bool $active = true)`. Верните ассоциативный массив с этими ключами. Вызовите с одним, двумя и тремя аргументами.

## Вариант 16
Реализуйте функцию `sumArray(array $arr): int|float`, возвращающую сумму элементов. Вызовите для [1, 2, 3, 4, 5], выведите 15.

## Вариант 17
Arrow function в usort: отсортируйте массив [3, 1, 4, 1, 5] по убыванию с помощью usort и fn($a, $b) => $b <=> $a. Выведите результат.

## Вариант 18
Функция с nullable: `formatDate(?int $timestamp): string`. Если null — верните 'N/A', иначе date('Y-m-d', $timestamp). Вызовите с time() и с null.

## Вариант 19
Рекурсия: функция factorial(int $n): int. Для n <= 0 верните 1, иначе n * factorial(n-1). Выведите factorial(5) = 120.

## Вариант 20
Функция `config(string $key, mixed $default = null): mixed`. Симулируйте массив конфигурации $config внутри функции (или передайте). Верните $config[$key] или $default. Вызовите config('host', 'localhost') и config('missing', 0). Выведите оба результата.
