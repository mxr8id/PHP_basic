# Задачи по теме: Циклы WHILE в PHP (20 вариантов)

## Вариант 1
Подсчитайте факториал числа N (например, 5) с помощью цикла while. Выведите 5! = 120.

## Вариант 2
Реализуйте «угадай число»: загадайте число 7. Цикл while(true), счётчик попыток. При «совпадении» (когда счётчик равен 7) выведите «Угадано с попытки N» и break.

## Вариант 3
Найдите первое число в диапазоне 1–100, делящееся на 7. Цикл while с переменной $i. Выведите это число (7).

## Вариант 4
Сравните while и do-while: начальное условие false ($x = 10; условие $x < 5). Для while тело не выполнится. Для do-while выполнится один раз. Выведите сообщение в каждом цикле и продемонстрируйте разницу.

## Вариант 5
Сумма чисел от 1 до 50 с помощью while. Переменные $sum, $i. Выведите результат (1275).

## Вариант 6
Цикл while: выводите квадраты чисел 1, 2, 3, … пока квадрат не превысит 100. Выведите последнее число и его квадрат.

## Вариант 7
Подсчитайте количество итераций: while с условием $i < 10, $i увеличивается на 2. Выведите количество итераций (5).

## Вариант 8
Найдите наименьшее число n, такое что 2^n > 1000. Цикл while, переменная $n и $value = 2**$n. Выведите n и value.

## Вариант 9
Цикл while с двумя переменными: $a = 0, $b = 10. Условие $a < $b. На каждой итерации a++, b--. Выводите пару (a, b). Сколько итераций?

## Вариант 10
Выведите числа от 5 до 1 с помощью while. Переменная $x = 5, условие $x >= 1, в конце $x--.

## Вариант 11
Симуляция «повтор до успеха»: переменная $success = false, счётчик попыток. В цикле while(!$success) увеличивайте счётчик; после 3 попыток присвойте success = true и выйдите. Выведите «Успех с попытки 3».

## Вариант 12
Сумма чётных чисел от 2 до 20 с помощью while. Выведите результат (110).

## Вариант 13
Цикл while: читайте «ввод» из массива [1, 2, 0, 4] по одному элементу (индекс 0, 1, 2…). Выход при значении 0. Выведите сумму прочитанных до нуля (3).

## Вариант 14
Найдите количество цифр в числе 12345 с помощью while: делить на 10, пока число > 0. Счётчик — количество цифр (5).

## Вариант 15
Выведите степени тройки (3^0, 3^1, 3^2, …), пока значение не станет больше 100. while, переменная $p = 1, затем * 3.

## Вариант 16
Цикл while с break: перебирайте числа от 1 до 20, выходите при первом числе, делящемся на 11 (это 11). Выведите это число.

## Вариант 17
Подсчитайте, сколько раз число 100 можно поделить на 2 (целочисленно), пока результат >= 1. while, счётчик. Ответ: 7 (100, 50, 25, 12, 6, 3, 1).

## Вариант 18
Числа Фибоначчи: выведите первые 8 чисел (1, 1, 2, 3, 5, 8, 13, 21) с помощью while. Две переменные для текущего и предыдущего.

## Вариант 19
Цикл while: массив [10, 20, 30, 40], индекс $i = 0. Выводите элемент, пока индекс < count. Выведите все четыре числа.

## Вариант 20
«Обратный отсчёт»: $t = 5. while ($t > 0) выводите $t, затем $t--. В конце выведите «Старт!». Без sleep, просто вывод 5, 4, 3, 2, 1, Старт!.
