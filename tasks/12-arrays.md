# Задачи по теме: Создание массивов в PHP (20 вариантов)

## Вариант 1
Создайте массив из 5 имён. Добавьте ещё 2 имени (через [] и array_push). Удалите первый (array_shift) и последний (array_pop) элемент. Выведите итоговый массив.

## Вариант 2
Проверьте наличие значения в массиве: массив [1, 2, 3, 4, 5], искомое 3. Используйте in_array. Выведите true/false. Выведите отдельно ключи (array_keys) и значения (array_values) исходного массива.

## Вариант 3
Объедините два массива: [1, 2, 3] и [4, 5, 6] через array_merge. Выведите результат. Затем объедините ассоциативные ['a' => 1] и ['b' => 2], выведите.

## Вариант 4
Создайте многомерный массив: массив пользователей, каждый элемент — массив с полями name, age. Минимум 3 пользователя. Переберите и выведите «Имя: X, Возраст: Y».

## Вариант 5
Создайте пустой массив. В цикле добавьте в него числа от 1 до 10 (через $arr[] = $i). Выведите count($arr) и последний элемент.

## Вариант 6
Удалите элемент по ключу: массив ['a' => 1, 'b' => 2, 'c' => 3], удалите 'b' (unset). Выведите массив. Переиндексируйте числовые ключи через array_values (для числового массива после unset).

## Вариант 7
Проверьте, существует ли ключ 'email' в массиве $user = ['name' => 'John', 'email' => 'j@mail.ru']. Используйте array_key_exists. Выведите результат. Добавьте ключ 'phone', выведите массив.

## Вариант 8
Объедините три массива в один: [1, 2], [3], [4, 5, 6]. Используйте array_merge. Выведите длину и сумму элементов (array_sum).

## Вариант 9
Создайте массив строк. Проверьте in_array с третьим параметром true (строгое сравнение): ищите число 0 и строку "0". Продемонстрируйте разницу.

## Вариант 10
Извлеките первый элемент массива [10, 20, 30, 40] через array_shift. Выведите извлечённое значение (10) и изменённый массив [20, 30, 40].

## Вариант 11
Добавьте в начало массива элемент с помощью array_unshift. Исходный массив [2, 3, 4], добавьте 1. Выведите [1, 2, 3, 4].

## Вариант 12
Создайте массив с дубликатами [1, 2, 2, 3, 3, 3]. Удалите дубликаты (array_unique). Выведите результат и количество элементов до и после.

## Вариант 13
Объедините ключи и значения в пары: массив ['x' => 10, 'y' => 20]. Сформируйте массив строк ['x=10', 'y=20'] через цикл или array_map. Выведите.

## Вариант 14
Проверьте, пустой ли массив. Функция или выражение: empty($arr) и count($arr) === 0. Проверьте для [] и [1, 2]. Выведите результаты.

## Вариант 15
Создайте массив диапазона чисел от 5 до 15 (range(5, 15)). Выведите количество и сумму элементов. Выведите первый и последний элемент (reset, end или индексы).

## Вариант 16
Объедините два ассоциативных массива с общим ключом: ['a' => 1, 'b' => 2] и ['b' => 20, 'c' => 3]. Выведите array_merge — второй перезапишет 'b'. Продемонстрируйте.

## Вариант 17
Получите срез массива: [0, 1, 2, 3, 4, 5], элементы с индекса 2 по 4 (array_slice). Выведите [2, 3, 4]. Продемонстрируйте сохранение ключей (третий параметр true).

## Вариант 18
Создайте массив из 5 нулей (array_fill(0, 5, 0)). Замените значения в цикле на 1, 2, 3, 4, 5. Выведите массив.

## Вариант 19
Проверьте, что array_push возвращает новую длину, а не массив. $a = [1, 2]; $len = array_push($a, 3, 4); Выведите $len (4) и $a [1,2,3,4].

## Вариант 20
Создайте массив строк. Добавьте элемент в конец. Удалите элемент по индексу 1 (unset). Выведите массив. Восстановите числовые индексы через array_values и выведите снова.
