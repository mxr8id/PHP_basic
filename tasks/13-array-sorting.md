# Задачи по теме: Сортировка массивов в PHP (20 вариантов)

## Вариант 1
Отсортируйте массив чисел [3, 1, 4, 1, 5, 9, 2, 6] по возрастанию (sort) и по убыванию (rsort). Выведите оба результата.

## Вариант 2
Ассоциативный массив ['Иван' => 30, 'Мария' => 25, 'Пётр' => 35]. Отсортируйте по значению с сохранением ключей (asort). Выведите. Затем по ключу (ksort), выведите.

## Вариант 3
Массив элементов ['name' => ..., 'age' => ...] (3 элемента). Отсортируйте по полю age по возрастанию с помощью usort и оператора <=>. Выведите отсортированный массив.

## Вариант 4
Используйте reset(), next(), current() для обхода массива [10, 20, 30]. Выведите текущий элемент после reset, после next (дважды), после next (третий). Покажите перемещение внутреннего указателя.

## Вариант 5
Отсортируйте массив строк ['банан', 'яблоко', 'вишня'] по возрастанию (sort). Выведите. Затем по убыванию (rsort).

## Вариант 6
Реализуйте сортировку по двум полям: массив записей с полями 'surname' и 'name'. Сначала по фамилии, при равенстве — по имени. usort с функцией сравнения (strcmp или <=> для строк).

## Вариант 7
Массив чисел. Найдите минимальный и максимальный элемент с помощью сортировки: отсортируйте копию (чтобы не менять исходный), выведите первый и последний элемент отсортированной копии. Альтернатива: min/max — но задание через сортировку.

## Вариант 8
Используйте uasort для сортировки ассоциативного массива по значению с сохранением ключей. Сравнение через пользовательскую функцию (например, по длине строки значения). Выведите результат.

## Вариант 9
Обойдите массив [1, 2, 3, 4, 5] с помощью current/next в цикле while. Условие: current() !== false. Выведите каждый элемент. После цикла вызовите reset() и выведите current() снова.

## Вариант 10
Отсортируйте массив по длине строк: ['короткий', 'очень длинная строка', 'средний']. usort с функцией strlen($a) <=> strlen($b). Выведите.

## Вариант 11
Массив с отрицательными и положительными числами. Отсортируйте по абсолютному значению (usort, сравнение abs($a) <=> abs($b)). Выведите.

## Вариант 12
Используйте end() и prev(). Массив из 4 элементов. Выведите последний (end), затем prev — предпоследний, prev — третий с конца. Выведите ключи и значения.

## Вариант 13
Сортировка по ключу в обратном порядке: krsort. Массив ['c' => 3, 'a' => 1, 'b' => 2]. Выведите после krsort (ключи c, b, a).

## Вариант 14
Массив дат в формате 'Y-m-d'. Отсортируйте по дате (арортировка строк дат подходит). sort или usort с strtotime для сравнения. Выведите от самой ранней к поздней.

## Вариант 15
Перемешайте массив [1, 2, 3, 4, 5] (shuffle). Выведите результат (каждый раз разный). Затем отсортируйте обратно (sort) и выведите.

## Вариант 16
Сортировка многомерного массива по вложенному ключу: например, по ['data']['score']. usort с $a['data']['score'] <=> $b['data']['score']. Создайте тестовый массив и выведите.

## Вариант 17
Используйте array_reverse после sort, чтобы получить массив по убыванию, не меняя исходный массив (сначала копия, sort, reverse). Выведите исходный и «по убыванию».

## Вариант 18
Найдите второй по величине элемент в массиве чисел. Отсортируйте (rsort), возьмите элемент с индексом 1. Выведите.

## Вариант 19
uksort — сортировка по ключам пользовательской функцией. Массив с ключами 'a', 'aa', 'b'. Отсортируйте по длине ключа (strlen). Выведите.

## Вариант 20
Объедините две сортировки: сначала отфильтруйте элементы > 10, затем отсортируйте отфильтрованный массив. Исходный [15, 5, 20, 8, 25]. Результат [15, 20, 25]. array_filter, затем sort или rsort по заданию.
