# Задачи по теме: Оператор ELSE в PHP (20 вариантов)

## Вариант 1
По числу от 1 до 7 выведите название дня недели. Используйте switch. Для 8 и более выведите «Неверный день».

## Вариант 2
Реализуйте то же (день недели по числу 1–7), но с помощью match. Верните строку и выведите её.

## Вариант 3
Калькулятор: переменные `$a`, `$b`, `$operator` ('+', '-', '*', '/'). Вычислите результат с помощью match и выведите. Для неизвестного оператора выведите «Ошибка».

## Вариант 4
Оценка по баллам: 90+ — «Отлично», 70–89 — «Хорошо», 50–69 — «Удовлетворительно», иначе — «Неуд». Реализуйте через if-elseif-else. Выведите для баллов 95, 75, 55, 30.

## Вариант 5
Напишите функцию `getSeason(int $month): string`, возвращающую «зима», «весна», «лето» или «осень». Используйте match. Для месяца вне 1–12 верните «Ошибка».

## Вариант 6
По коду страны (ru, en, de, fr) выведите полное название. Используйте switch с break. Добавьте default: «Неизвестная страна».

## Вариант 7
Реализуйте то же (название страны по коду) через match. Сохраните результат в переменную и выведите.

## Вариант 8
Переменная `$role`: 'admin', 'user', 'guest'. Выведите уровень доступа: 3, 2, 1 соответственно. Используйте if-elseif-else.

## Вариант 9
По числу месяца (1–12) определите квартал (1–3, 4–6, 7–9, 10–12). Используйте match, возвращающий «Q1», «Q2», «Q3», «Q4».

## Вариант 10
Оценка студента: буквы A, B, C, D, F. Переведите в текст: «Отлично», «Хорошо», «Удовлетворительно», «Плохо», «Неуд». Используйте switch.

## Вариант 11
Переменная `$status`: 'new', 'pending', 'done', 'cancelled'. Выведите цвет для статуса (например, green для done). Используйте match.

## Вариант 12
По возрасту выведите категорию: 0–2 «младенец», 3–12 «ребёнок», 13–17 «подросток», 18+ «взрослый». if-elseif-else.

## Вариант 13
Код валюты: 'USD', 'EUR', 'RUB'. Верните символ: $, €, ₽. Используйте match. Для неизвестной валюты — '?'.

## Вариант 14
Число от 1 до 5 — оценка. Выведите текстом: «единица», «двойка», …, «пятёрка». switch с default «неверная оценка».

## Вариант 15
Тип фигуры: 'circle', 'square', 'triangle'. По типу выведите формулу площади (строка, например «πr²»). match.

## Вариант 16
Переменная `$hour` (0–23). Выведите «ночь» (0–5), «утро» (6–11), «день» (12–17), «вечер» (18–23). if-elseif-else.

## Вариант 17
По коду ошибки (404, 403, 500) выведите сообщение: «Не найдено», «Доступ запрещён», «Ошибка сервера». switch.

## Вариант 18
Статус заказа: 0 — новый, 1 — в обработке, 2 — отправлен, 3 — доставлен. Выведите текст статуса. match с default.

## Вариант 19
Пол (m/f). Выведите «Мужской» или «Женский». Учтите регистр (strtolower). if-else.

## Вариант 20
Уровень логирования: 'debug', 'info', 'warning', 'error'. Выведите числовой приоритет (1, 2, 3, 4). match. Неизвестный — 0.
