# Лекции по языку PHP

Курс лекций по основам PHP 8.x для начинающих разработчиков.

## Требования

- PHP 8.0+
- Совместимость с Laravel Sail

## Структура проекта

```
PHP_basic/
├── README.md
├── lectures/           # Markdown-файлы лекций
│   ├── 01-variables.md
│   ├── 02-types.md
│   └── ...
├── examples/           # Рабочие примеры кода
│   ├── 01-variables/
│   ├── 02-types/
│   └── ...
└── tasks/              # Задачи по темам (20 вариантов на каждую лекцию)
    ├── 01-variables.md
    ├── 02-types.md
    └── ...
```

## Структура каждой лекции

1. **Теоретическая часть** — основные концепции
2. **Синтаксис и основные концепции**
3. **Рабочие примеры с пояснениями**
4. **Практические задания** (минимум 3 на лекцию)
5. **Частые ошибки и их решение**

## Запуск примеров

```bash
# Через PHP CLI
php examples/01-variables/basic.php

# Через Laravel Sail (если используется)
sail php examples/01-variables/basic.php
```

## Задачи

В папке `tasks/` для каждой лекции подготовлено **20 вариантов** заданий. Файлы именуются по номеру лекции: `01-variables.md`, `02-types.md` и т.д. Каждый вариант — отдельное практическое задание по теме.
