# Лекция 5: Логические операторы PHP

## 1. Теоретическая часть

### Логические операторы

- `&&` / `and` - логическое И (разный приоритет)
- `||` / `or` - логическое ИЛИ
- `!` - отрицание
- `xor` - исключающее ИЛИ (true если только один операнд true)

### Приоритет операторов

`&&` и `||` имеют более высокий приоритет, чем `and` и `or`. Рекомендуется использовать `&&` и `||`.

### Короткое замыкание (short-circuit evaluation)

При `$a && $b`: если `$a` ложно, `$b` не вычисляется. При `$a || $b`: если `$a` истинно, `$b` не вычисляется. Это важно для вызовов функций и проверок.

---

## 2. Синтаксис и основные концепции

```php
$a && $b   // И (короткое замыкание)
$a || $b   // ИЛИ (короткое замыкание)
!$a        // Отрицание
$a ?? $b   // Null coalescing (PHP 7+)
$a ?: $b   // Elvis: $a если truthy, иначе $b
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Неожиданный результат с and/or | Низкий приоритет and, or | Используйте `&&`, `||` или скобки |
| Побочные эффекты в условии | Функция вызывается при коротком замыкании | Выносите вызов в переменную до условия |
| Смешивание & и && | `&` - побитовое И | Для логики используйте `&&` |
| Проверка на null | `!$var` при $var=0 | Используйте `$var === null` или `isset()` |
