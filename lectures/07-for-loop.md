# Лекция 7: Цикл FOR в PHP

## 1. Теоретическая часть

### Синтаксис цикла for

```php
for (инициализация; условие; инкремент) { тело }
```

- **Инициализация** - выполняется один раз в начале
- **Условие** - проверяется перед каждой итерацией
- **Инкремент** - выполняется после каждой итерации

### Вложенные циклы

Цикл внутри цикла. Используется для обхода многомерных массивов, таблиц, матриц.

### Управление циклом

- `break` - выход из цикла
- `continue` - переход к следующей итерации
- `break N` / `continue N` - выход/пропуск N уровней вложенных циклов

---

## 2. Синтаксис и основные концепции

```php
for ($i = 0; $i < 10; $i++) { }
for ($i = 0, $j = 10; $i < $j; $i++, $j--) { }
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Бесконечный цикл | Условие всегда true | Проверьте инкремент и условие выхода |
| Off-by-one | `$i <= 10` вместо `$i < 10` | Уточните, нужны ли границы включительно |
| Изменение счётчика в теле | `$i--` или `$i += 2` в теле | Учитывайте при отладке |
| Пустое тело цикла | Точка с запятой после for | Уберите `;` или добавьте тело |
