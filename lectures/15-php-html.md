# Лекция 15: Использование PHP в связке с HTML

## 1. Теоретическая часть

### Встраивание PHP в HTML

- Теги `<?php ... ?>` и короткий вывод `<?= ... ?>`
- PHP обрабатывается на сервере, клиент получает только HTML

### Переменные в шаблонах

- Прямой вывод: `<?= $variable ?>`
- Всегда экранируйте: `<?= htmlspecialchars($variable, ENT_QUOTES, 'UTF-8') ?>`

### Условия и циклы в шаблонах

- Альтернативный синтаксис: `<?php if (...): ?> ... <?php endif; ?>`
- Улучшает читаемость в смешанном HTML/PHP

### Безопасность

- Защита от XSS через `htmlspecialchars()`
- Null coalescing для отсутствующих значений: `$_POST['name'] ?? ''`

---

## 2. Синтаксис и основные концепции

```php
<?= htmlspecialchars($var) ?>
<?php foreach ($items as $item): ?>
    <li><?= htmlspecialchars($item) ?></li>
<?php endforeach; ?>
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| XSS | Неэкранированный вывод | Всегда htmlspecialchars() |
| Undefined index | Отсутствующий ключ в $_POST | Используйте ?? '' или isset() |
| Короткие теги <?= | Отключены short_open_tag | <?= обычно работает в PHP 5.4+ |
| Смешение логики и представления | Вся логика в шаблоне | Выносите в отдельные файлы/функции |
