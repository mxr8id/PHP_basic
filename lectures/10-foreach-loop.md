# Лекция 10: Цикл FOREACH в PHP

## 1. Теоретическая часть

### Перебор массивов

`foreach` предназначен для итерации по массивам и объектам, реализующим `Traversable`.

### Синтаксис

```php
foreach ($array as $value) { }
foreach ($array as $key => $value) { }
```

### Перебор по ссылке

При использовании `foreach ($array as &$value)` изменения `$value` изменяют исходный массив. После цикла рекомендуется `unset($value)` во избежание неожиданного поведения.

---

## 2. Синтаксис и основные концепции

```php
foreach ($arr as $v) { }
foreach ($arr as $k => $v) { }
foreach ($arr as &$v) { $v *= 2; } unset($v);
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Modification during iteration | Изменение массива в foreach | Итерируйте по копии или собирайте изменения |
| Ссылка после цикла | `&$value` остаётся | Вызовите `unset($value)` после цикла |
| foreach не по массиву | Передана не итерируемая переменная | Проверяйте `is_array()` |
| Ошибка при null | foreach(null) в PHP 7 | В PHP 8 - TypeError, проверяйте заранее |
