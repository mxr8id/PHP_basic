# Лекция 17: Проверка данных формы с помощью PHP

## 1. Теоретическая часть

### Валидация данных

- Проверка на пустоту, длину, формат
- Сбор ошибок в массив с привязкой к полям
- Отображение ошибок рядом с полями формы

### Фильтрация данных

- `filter_var()` с FILTER_VALIDATE_* и FILTER_SANITIZE_*
- `htmlspecialchars()` при выводе
- Приведение типов `(int)`, `(float)` для числовых полей

### Защита от инъекций

- Подготовленные запросы для SQL
- htmlspecialchars для XSS
- Проверка типов и границ для числовых значений

### Отображение ошибок

- Массив `$errors['field']` для каждого поля
- Вывод в шаблоне рядом с input
- Общий блок с списком ошибок (опционально)

---

## 2. Синтаксис и основные концепции

```php
$name = trim($_POST['name'] ?? '');
if (empty($name)) $errors['name'] = "Обязательно";
$email = filter_var($_POST['email'] ?? '', FILTER_VALIDATE_EMAIL);
$age = (int)($_POST['age'] ?? 0);
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| trim не применяется | Проверка до trim | Сначала trim, потом empty/длина |
| filter_var с пустой строкой | Возвращает false | Проверяйте empty отдельно |
| Неверные границы | Возраст 0 проходит | Явная проверка $age >= 18 && $age <= 100 |
| XSS в ошибках | Вывод ошибки без экранирования | htmlspecialchars($errors['name']) |
