# Лекция 20: Сессии в PHP

## 1. Теоретическая часть

### Работа с сессиями

- `session_start()` - запуск сессии (до любого вывода!)
- `$_SESSION` - массив для хранения данных сессии
- Данные сохраняются на сервере, идентификатор передаётся через cookie

### Настройка сессий

- `session_set_cookie_params()` - параметры cookie
- `ini_set('session.cookie_httponly', 1)` - защита от XSS
- `session_regenerate_id(true)` - защита от фиксации сессии

### Безопасность сессий

- HttpOnly, Secure, SameSite для cookie
- Регенерация ID при смене прав (логин)
- Таймаут неактивности (реализуется вручную)

### Управление данными сессии

- `session_unset()` - очистка переменных
- `session_destroy()` - уничтожение сессии
- `session_write_close()` - ранняя запись (освобождение блокировки)

---

## 2. Синтаксис и основные концепции

```php
session_start();
$_SESSION['user_id'] = 123;
session_regenerate_id(true);
session_destroy();
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Headers already sent | Вывод до session_start | session_start() в самом начале |
| Сессия не сохраняется | Cookie не отправляются | Проверьте path, domain, SameSite |
| Фиксация сессии | Один ID до и после логина | session_regenerate_id(true) после логина |
| Блокировка | Долгое выполнение скрипта | session_write_close() после чтения |
