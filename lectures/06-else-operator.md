# Лекция 6: Оператор ELSE в PHP

## 1. Теоретическая часть

### if-else конструкции

- `if (условие) { ... }` - выполнить, если условие истинно
- `else { ... }` - выполнить, если условие ложно
- `elseif` / `else if` - дополнительное условие

### switch-case

Множественный выбор по значению. Важно не забывать `break`, иначе выполнение «провалится» в следующий case.

### Match expression (PHP 8.0+)

Современная замена switch: выражение, возвращающее значение. Строгое сравнение (===), обязательное покрытие всех случаев или `default`.

---

## 2. Синтаксис и основные концепции

```php
if ($x > 0) { } elseif ($x < 0) { } else { }
switch ($x) { case 1: break; default: }
$result = match($x) { 1 => 'a', 2 => 'b', default => 'c' };
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Провал в switch | Отсутствие break | Добавьте `break` или используйте match |
| Нет default в match | Непокрытый случай | Добавьте `default =>` |
| Присваивание в условии | `if ($x = 5)` вместо `==` | Используйте `===` для сравнения |
| Сравнение строк в switch | Неточное совпадение | Учитывайте регистр или используйте `strtolower()` |
