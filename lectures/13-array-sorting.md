# Лекция 13: Сортировка массивов в PHP

## 1. Теоретическая часть

### Сортировка по значению

- `sort()` - по возрастанию (сбрасывает ключи)
- `rsort()` - по убыванию
- `asort()` - по значению, сохраняя ключи
- `arsort()` - по убыванию, сохраняя ключи

### Сортировка по ключу

- `ksort()` - по ключам по возрастанию
- `krsort()` - по ключам по убыванию

### Пользовательские функции

- `usort()` - пользовательское сравнение (по значению)
- `uasort()` - с сохранением ключей
- `uksort()` - по ключам

### Навигация по массиву

- `reset()`, `end()` - указатель в начало/конец
- `current()`, `next()`, `prev()` - текущий, следующий, предыдущий элемент

---

## 2. Синтаксис и основные концепции

```php
sort($arr);
asort($arr);
usort($arr, fn($a, $b) => $a <=> $b);
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Ключи сброшены после sort | sort переиндексирует | Используйте asort для ассоциативных |
| Модификация внутри usort | Нестабильное сравнение | Сравнивайте по одному полю или стабильно |
| Указатель массива | current/next двигают указатель | После цикла reset() при необходимости |
| Строковая сортировка | Локаль и регистр | Используйте strcoll или strcasecmp |
