# Лекция 1: Переменные в PHP

## 1. Теоретическая часть

### Синтаксис объявления переменных

В PHP переменные объявляются с помощью символа `$`, за которым следует имя переменной. PHP - язык с динамической типизацией: тип переменной определяется значением, которое в неё присваивается.

### Соглашения об именовании

- Имена переменных чувствительны к регистру (`$name` и `$Name` - разные переменные)
- Имя должно начинаться с буквы или подчёркивания `_`
- Рекомендуется camelCase: `$userName`, `$isActive`
- Константы традиционно в UPPER_SNAKE_CASE: `MAX_USERS`, `PI`

### Область видимости переменных

- **Локальные** - внутри функции, видны только в ней
- **Глобальные** - объявлены вне функций, видны в основном скрипте
- **Суперглобальные** - `$_GET`, `$_POST`, `$_SESSION` и др. - доступны везде
- **Статические** - сохраняют значение между вызовами функции

### Константы

Константы объявляются через `const` (на уровне класса или в глобальной области) или `define()` (в любом месте). После объявления их значение изменить нельзя.

---

## 2. Синтаксис и основные концепции

```php
$variableName = value;    // Объявление с присваиванием
const CONST_NAME = value; // Константа класса/глобальная
define('CONST_NAME', value); // Константа через define()
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `Undefined variable` | Использование переменной до присваивания | Инициализируйте переменную или используйте null coalescing: `$x ?? 'default'` |
| `Constant already defined` | Повторное определение константы | Проверяйте через `defined('CONST_NAME')` перед `define()` |
| Изменение константы | Попытка присвоить новое значение | Константы неизменяемы, используйте переменную |
| Опечатка в имени | `$usre` вместо `$user` | Используйте осмысленные имена и IDE с автодополнением |
