# Лекция 3: Операторы PHP

## 1. Теоретическая часть

### Математические операторы

- `+` - сложение
- `-` - вычитание
- `*` - умножение
- `/` - деление (всегда float, даже при делении целых)
- `%` - остаток от деления
- `**` - возведение в степень (PHP 5.6+)

### Строковые операторы

- `.` - конкатенация строк

### Математические функции

Встроенные функции: `abs()`, `sqrt()`, `round()`, `ceil()`, `floor()`, `max()`, `min()`, `rand()` и др.

---

## 2. Синтаксис и основные концепции

```php
$a + $b   // Сложение
$a . $b   // Конкатенация строк
$a ** $b  // Степень
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Деление на ноль | `$a / 0` | Проверяйте делитель перед делением |
| Неверный тип при сложении | Строка + число | PHP приводит типы, но лучше явно: `(int)$str` |
| Остаток с float | `%` с дробными числами | Используйте `fmod()` для float |
| Переполнение int | Слишком большое число | Используйте GMP или BC Math |
