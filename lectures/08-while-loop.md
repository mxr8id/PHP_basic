# Лекция 8: Циклы WHILE в PHP

## 1. Теоретическая часть

### Цикл while

Условие проверяется **до** выполнения тела. Если условие изначально ложно, тело не выполнится ни разу.

```php
while (условие) { тело }
```

### Цикл do-while

Тело выполняется минимум один раз. Условие проверяется **после** выполнения тела.

```php
do { тело } while (условие);
```

### Различия между циклами

| Цикл | Проверка условия | Минимум итераций |
|------|------------------|------------------|
| while | До тела | 0 |
| do-while | После тела | 1 |
| for | До тела | 0 |

---

## 2. Синтаксис и основные концепции

```php
while ($condition) { }
do { } while ($condition);
while (true) { if ($exit) break; }
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Бесконечный цикл | Условие не меняется | Обновляйте переменные в условии внутри тела |
| readline в CLI | В Web нет readline | Используйте другие способы ввода в Web |
| Слишком много итераций | Неверное условие выхода | Добавьте счётчик или таймаут |
