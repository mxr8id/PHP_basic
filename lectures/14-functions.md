# Лекция 14: Функции в PHP

## 1. Теоретическая часть

### Синтаксис функций

```php
function name(параметры): returnType { тело }
```

### Параметры и возвращаемые значения

- Обязательные и опциональные параметры
- Значения по умолчанию
- Типизация параметров и возвращаемого типа

### Анонимные функции (closures)

- `function ($x) { return $x * 2; }`
- Использование `use` для замыкания переменных из внешней области

### Arrow functions (PHP 7.4+)

- `fn($a, $b) => $a + $b`
- Всегда возвращают значение, поддерживают только одно выражение

### Variadic-параметры

- `...$args` - произвольное количество аргументов

---

## 2. Синтаксис и основные концепции

```php
function greet(string $name): string { return "Hello, $name"; }
$fn = fn($x) => $x * 2;
function sum(...$nums) { return array_sum($nums); }
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Опциональные до обязательных | Синтаксическая ошибка | Сначала обязательные, потом опциональные |
| Изменение по ссылке | Неявная мутация | Явно указывайте `&$param` при необходимости |
| return без значения | В типизированной функции | Возвращайте корректный тип или void |
| use в closure | Захват по значению | Для изменения используйте `use (&$var)` |
