# Лекция 9: Цикл DO...WHILE в PHP

## 1. Теоретическая часть

### Синтаксис do-while

```php
do {
    // тело выполняется минимум один раз
} while (условие);
```

### Различия с циклом while

- **while**: условие проверяется до входа в тело
- **do-while**: условие проверяется после выполнения тела

### Практическое применение

Используется, когда нужно гарантированно выполнить действие хотя бы один раз: ввод с валидацией, повторные попытки подключения, меню и т.д.

## 2. Синтаксис и основные концепции

```php

    // код
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Бесконечный цикл | Условие всегда true | Убедитесь, что переменная в условии обновляется |
| Точка с запятой | Пропуск `;` после while | Синтаксис: `} while (...);` |
| readline в Web | readline только для CLI | Используйте формы/GET для Web |
