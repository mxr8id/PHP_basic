# Лекция 4: Операторы сравнения PHP

## 1. Теоретическая часть

### Сравнение значений и типов

- `==` - равенство (с приведением типов)
- `===` - идентичность (значение и тип)
- `!=` / `<>` - неравенство
- `!==` - не идентичность
- `<`, `>`, `<=`, `>=` - сравнение порядка
- `<=>` - spaceship operator (PHP 7+): возвращает -1, 0 или 1

### Важность строгого сравнения

Использование `===` предотвращает неожиданное приведение типов (например, `"0" == false` даёт true).

---

## 2. Синтаксис и основные концепции

```php
$a == $b   // Равны (с приведением)
$a === $b  // Идентичны (тип + значение)
$a <=> $b  // -1 если a<b, 0 если a==b, 1 если a>b
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Строка "0" == false | Приведение типов | Используйте `===` |
| Сравнение float | Погрешность представления | Используйте допуск: `abs($a - $b) < 0.00001` |
| Сравнение объектов | Разные экземпляры | Реализуйте `__toString()` или сравнивайте свойства |
| Сортировка строк | `strcmp()` для locale | `strcoll()` или `strcmp()` в зависимости от задачи |
