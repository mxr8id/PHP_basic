# Лекция 18: Взаимодействие PHP и MySQL

## 1. Теоретическая часть

### Современные методы подключения

- **PDO** (PHP Data Objects) - рекомендуемый подход, поддержка нескольких СУБД
- **mysqli** - нативная расширение только для MySQL/MariaDB

### Подготовленные запросы (Prepared Statements)

- Защита от SQL-инъекций
- Placeholders: `?` или именованные `:name`
- Привязка параметров через `bindValue()` / `bindParam()`

### Безопасность соединения

- Хранение учётных данных вне репозитория (env-файлы)
- Использование charset utf8mb4
- Обработка исключений при подключении

### Обработка ошибок

- PDO: `PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION`
- try-catch для PDOException

---

## 2. Синтаксис и основные концепции

```php
$pdo = new PDO($dsn, $user, $pass, [PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION]);
$stmt = $pdo->prepare("SELECT * FROM users WHERE id = ?");
$stmt->execute([$id]);
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Пароль в коде | Утечка при коммите | Используйте .env и getenv() |
| SQL-инъекция | Конкатенация в запрос | Всегда prepared statements |
| Charset проблемы | Кириллица искажена | utf8mb4 в DSN и set_charset |
| Connection refused | MySQL не запущен | Проверьте Sail/docker или локальный MySQL |
