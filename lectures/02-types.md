# Лекция 2: Типы данных PHP

## 1. Теоретическая часть

### Скалярные типы (int, float, string, bool)

- **int** - целые числа
- **float** - числа с плавающей точкой
- **string** - строки (одинарные или двойные кавычки)
- **bool** - логический тип (true/false)

### Составные типы (array, object)

- **array** - индексированные или ассоциативные массивы
- **object** - экземпляры классов

### Специальные типы (null, resource)

- **null** - отсутствие значения
- **resource** - ссылка на внешний ресурс (файл, БД и т.д.)

### Типизация в современном PHP

С PHP 7.0+ поддерживаются объявления типов для параметров и возвращаемых значений. PHP 8.0 добавил union types, mixed, static return type.

## 2. Синтаксис и основные концепции

```php
// Скалярные
$int = 42;
$float = 3.14;
$string = "Hello";
$bool = true;

// Составные
$array = [1, 2, 3];
$object = new stdClass();

// Функция с типизацией
function add(int $a, int $b): int { return $a + $b; }
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `TypeError` | Передан неверный тип | Добавьте валидацию или используйте union types |
| Неожиданное приведение | Строка "123" + число | Явно приводите типы: `(int)$str` |
| Сравнение типов | `==` игнорирует тип | Используйте строгое сравнение `===` |
| Пустая строка как false | `"" == false` даёт true | Используйте `===` или `empty()` осознанно |
