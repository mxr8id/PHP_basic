# Лекция 16: Получение данных от элементов формы с помощью PHP

## 1. Теоретическая часть

### Методы GET и POST

- **GET** - данные в URL, кэшируются, ограничение длины. Для поиска, фильтров.
- **POST** - данные в теле запроса, для форм, загрузки файлов.

### Суперглобальные массивы

- `$_GET` - параметры URL
- `$_POST` - данные из тела (method="post")
- `$_REQUEST` - GET + POST + COOKIE (не рекомендуется)
- `$_FILES` - загруженные файлы

### Безопасное получение данных

- Null coalescing: `$_POST['name'] ?? ''`
- Приведение типов: `(int)($_GET['page'] ?? 1)`
- Фильтрация: `filter_var()`, `filter_input()`

### Обработка файлов

- `$_FILES['name']['error']` - код ошибки
- `$_FILES['name']['tmp_name']` - временный путь
- `move_uploaded_file()` - перемещение в целевую папку

---

## 2. Синтаксис и основные концепции

```php
$name = $_POST['name'] ?? '';
$page = (int)($_GET['page'] ?? 1);
if ($_FILES['file']['error'] === UPLOAD_ERR_OK) { ... }
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Undefined array key | Отсутствующий ключ | ?? '' или isset() |
| Файл не загружается | Проверка error | Всегда проверяйте $_FILES['x']['error'] |
| Неверный путь | uploads/ не существует | mkdir или проверка is_dir |
| SQL-инъекции | Прямая подстановка в запрос | Используйте prepared statements |
