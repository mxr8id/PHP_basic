# Лекция 12: Создание массивов в PHP

## 1. Теоретическая часть

### Создание массивов

- Короткий синтаксис: `[1, 2, 3]`
- Функция: `array(1, 2, 3)`
- Ассоциативные: `['key' => 'value']`

### Добавление и удаление элементов

- `$arr[] = $value` - добавить в конец
- `array_push($arr, ...)` - добавить в конец (возвращает новую длину!)
- `array_pop($arr)` - извлечь последний
- `array_shift($arr)` - извлечь первый
- `unset($arr[$key])` - удалить элемент

### Полезные функции

- `count()` - количество элементов
- `in_array()` - проверка наличия значения
- `array_keys()`, `array_values()` - ключи и значения
- `array_merge()` - слияние массивов

---

## 2. Синтаксис и основные концепции

```php
$arr = [1, 2, 3];
$arr[] = 4;
array_push($arr, 5, 6); // Добавляет, возвращает count
$last = array_pop($arr);
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| array_push возвращает int | Возвращает новый count | Не присваивайте: `array_push($arr, $v)` |
| unset не переиндексирует | Ключи остаются | Используйте `array_values()` для переиндексации |
| in_array строгость | "5" найдёт 5 при loose | Третий параметр `true` для strict |
| Пустой массив при merge | Передача не массивов | Проверяйте типы аргументов |
