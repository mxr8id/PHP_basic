# Лекция 11: Функции для обработки строк в PHP

## 1. Теоретическая часть

### Базовые строковые функции

- `strlen()` - длина строки в байтах (для UTF-8 используйте `mb_strlen()`)
- `strtoupper()`, `strtolower()` - регистр
- `ucfirst()`, `ucwords()` - первая буква слова/строки

### Поиск и замена

- `strpos()`, `stripos()` - позиция подстроки
- `str_replace()`, `str_ireplace()` - замена
- `substr()` - подстрока (в PHP 8.0+ `mb_substr()` для многобайтовых)

### Работа с регистром

- `mb_strtolower()`, `mb_strtoupper()` - для UTF-8

### Манипуляции

- `explode()`, `implode()` - разбиение и объединение
- `trim()`, `ltrim()`, `rtrim()` - удаление пробелов
- `sprintf()`, `printf()` - форматирование

---

## 2. Синтаксис и основные концепции

```php
strlen($str);
strpos($haystack, $needle);
str_replace($search, $replace, $subject);
explode($delimiter, $string);
implode($glue, $array);
```

## 5. Частые ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| strpos возвращает 0 | Подстрока в начале | Проверяйте `!== false` |
| Кириллица в strlen | Байты, не символы | Используйте `mb_strlen()` |
| str_replace массивов | Замена нескольких значений | Оба параметра могут быть массивами |
| XSS в выводе | Неэкранированный вывод | Всегда `htmlspecialchars()` при выводе в HTML |
